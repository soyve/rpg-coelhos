"Cenourinhas Brabas" é um mini RPG feito em Python onde personagens — coelhinhos guerreiros — lutam usando rolagens de dados.
O jogo registra cada batalha em arquivos de log, permitindo análises posteriores como dano médio, vitórias, rounds e muito mais.
Um projeto simples, divertido e perfeito para treinar lógica de programação, Python e análise de dados.
